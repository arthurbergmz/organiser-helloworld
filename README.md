# HelloWorld
###### Organiser example
---

A server with only a GET endpoint at `localhost:3000` (default) that shows `Hello, world!` as plain text.

## how to run

1. install dependencies

```bash
$ npm install
```

2. compile scripts

```bash
$ npm run build
```

3. start server

```bash
$ npm start
```

## how to develop

1. install dependencies

```bash
$ npm install
```

2. start server (with hot reload)

```bash
$ npm run dev
```
